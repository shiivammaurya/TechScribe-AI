# 🚀 TechScribe AI

### Intelligent Technical Writing Assistant

TechScribe AI is a Python-based desktop application designed to improve writing quality through intelligent spell correction, technical vocabulary recognition, writing analytics, and quality assessment.

Unlike traditional spell checkers, TechScribe AI understands domain-specific terminology used in Programming, AI/ML, Business, and Academic writing, reducing false corrections and improving user productivity.

---

## 📌 Features

### ✅ Intelligent Spell Correction
- Detects spelling mistakes
- Generates corrected text automatically
- Preserves valid technical terms

### ✅ Domain-Specific Dictionaries
Supports specialized vocabularies for:

- Programming
- AI / Machine Learning
- Business
- Academic Writing

### ✅ Writing Analytics
Provides:

- Word Count
- Character Count
- Sentence Count
- Reading Time Estimation
- Vocabulary Assessment

### ✅ Writing Quality Score
Generates a writing score based on:
- Spelling accuracy
- Writing quality
- Content complexity

### ✅ Modern User Interface
Built using CustomTkinter with:
- Dark Theme UI
- Professional Dashboard Layout
- Responsive Analytics Panel
